### This project is a front-end shopping cart simulation created to practice and demonstrate core JavaScript concepts applied to a real-world scenario.
### It represents a practical introduction to how JavaScript adds logic, interactivity, and dynamic behavior to a web application.
### The project simulates a fictional figure store, Sakura Collectibles, where users can browse products, add them to a cart, and see prices update automatically.

![tumblr_428dd9db43928be26c55e87b53e58ff8_374e6206_540](https://github.com/user-attachments/assets/a8d9b911-4fba-47c5-a6d0-785f5e79acb5)

## Project Overview:
### The application is divided into two main parts:
#### Landing Page (Storefront): Displays collectible figures with images, names, and prices in a visually cute (kawaii :3) layout!
#### Shopping Cart Page: Allows users to add products, manage quantities, and view the total cost dynamically.
### The goal is to mimic the basic behavior of a real e-commerce cart while keeping the logic simple and easy to understand.

![tumblr_c8f93e4e4e4366d36b23d56f282eecde_ca25b796_540](https://github.com/user-attachments/assets/3d937ec0-3345-4bae-9967-fd533836527c)

## How Does the Project Work?
### Users interact with the interface through forms and buttons.
### With JavaScript, the application:
#### - Adds selected products to the cart.
#### - Calculates the total price automatically.
#### - Updates quantities and values in real time.
#### - Clear the cart when needed.
#### - Provide visual feedback through dynamic UI updates.
